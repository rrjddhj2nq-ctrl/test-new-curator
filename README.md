# test-curator

Steakhouse 风格的机构 DeFi 落地页（Create React App + TypeScript）。

## 线上地址

- Vercel：https://test-curator.vercel.app
- GitHub Pages：https://rrjddhj2nq-ctrl.github.io/test-curator/

国内 Vercel 可能超时，详见 [DEPLOY.md](./DEPLOY.md)。

## 开发

```bash
npm install
npm start
npm run build
```

## 维护

| 改什么 | 文件 |
|--------|------|
| 文案 | `src/data/sections.ts` |
| 品牌/导航 | `src/config/site.ts` |
| 样式 | `src/styles/site.css` |
